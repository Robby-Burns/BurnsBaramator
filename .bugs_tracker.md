## ‚úÖ Resolved Bugs

### BUGS-R05: Vercel Build Command Conflict
**Status:** Resolved
**Priority:** High
**Severity:** Blocker
**Found:** Feb 2026
**Fixed:** Feb 2026

**Problem:**
Vercel build failed with `error: externally-managed-environment` because `vercel.json` forced a `pip install` command while the environment was managed by `uv`.

**Solution:**
Simplified `vercel.json` to use `{"framework": "fastapi"}`. This allows Vercel to auto-detect the project structure and use `uv` natively to install dependencies from `pyproject.toml`.

### BUGS-R04: API Database Connection Failure
**Status:** Resolved
**Priority:** High
**Severity:** Major
**Found:** Feb 2026
**Fixed:** Feb 2026

**Problem:**
`api/index.py` failed because it used SQLite-specific methods (`cursor`, `fetchall`) on the new SQLAlchemy-based `DatabaseManager`.

**Solution:**
Updated `api/index.py` to use `db.engine.connect()` and `text()` queries.
